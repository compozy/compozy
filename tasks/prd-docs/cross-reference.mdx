## Usage Instructions

### When to Use Each Template

1. **Standard Cross-Reference Section**: Use on every documentation page for basic navigation
2. **Component-Specific Cross-References**: Use based on the primary focus of the page (agent, task, tool, memory)
3. **Learning Paths**: Include on overview pages and tutorial content
4. **Learning Path Breadcrumbs**: Use in sequential tutorial content
5. **See Also Section**: Use at the end of comprehensive guides and detailed documentation

### Customization Guidelines

1. **Replace placeholders** in `{BRACKETS}` with actual content
2. **Choose appropriate icons** (ü§ñ for agents, ‚öôÔ∏è for tasks, üîß for tools, üß† for memory)
3. **Use bidirectional arrows** (‚Üî) to show two-way relationships
4. **Include time estimates** for learning paths where appropriate
5. **Keep descriptions brief** but informative

### Consistency Requirements

1. **Always use the same section headers** ("Related Documentation", "Cross-References", etc.)
2. **Maintain bidirectional linking** - if page A links to page B, page B should link back to page A
3. **Update learning paths** when new documentation is added
4. **Use consistent URL patterns** and link text
5. **Test all links** before publishing

This template system ensures consistent cross-referencing patterns across all Compozy documentation while making it easy for maintainers to add proper navigation to new content.
