# Title: Implement State Helper Methods
# Status: done
# Dependencies: 3

# Description: Add convenience methods to State struct for identifying task types and relationships
# Details:
Implement helper methods on the State struct:
```go
func (s *State) IsParallelRoot() bool {
    return s.ParentStateID == nil && s.ExecutionType == "parallel"
}

func (s *State) IsChildTask() bool {
    return s.ParentStateID != nil
}

func (s *State) HasParent() bool {
    return s.ParentStateID != nil
}

func (s *State) GetParentID() *core.ID {
    return s.ParentStateID
}
```
These methods will simplify conditional logic throughout the codebase and improve readability.

# Test Strategy:
Unit tests for each helper method with various State configurations, test edge cases with nil ParentStateID, verify boolean logic correctness
