# ADR-003: Default order list page size is 20

- Status: Accepted
- Date: 2026-07-11

## Context

The orders list endpoint needs a default page size.

## Decision

Default to 20 items per page, capped at 100.

## Alternatives Considered

- 50 (rejected: heavier default payload).

## Consequences

- A tunable, feature-local default. No cross-feature implication.
