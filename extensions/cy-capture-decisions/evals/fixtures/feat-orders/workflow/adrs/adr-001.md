# ADR-001: Name order tables with an `ord_` prefix

- Status: Accepted
- Date: 2026-07-11

## Context

The orders feature adds several tables and we wanted a consistent, greppable naming scheme within the
feature.

## Decision

Prefix every orders table with `ord_` (`ord_order`, `ord_line_item`).

## Alternatives Considered

- No prefix (rejected: harder to grep within the feature).

## Consequences

- Local readability for this feature only. No cross-feature implications.
