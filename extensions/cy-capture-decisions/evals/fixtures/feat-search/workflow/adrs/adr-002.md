# ADR-002: BM25 relevance ranking

- Status: Accepted
- Date: 2026-07-11

## Context

Ranking quality matters for every search surface. The plan chose BM25.

## Decision

Rank results with BM25.

## Alternatives Considered

- TF-IDF (rejected: weaker on long documents).

## Consequences

- Cross-feature-durable ranking contract — IF it actually ships. (In this fixture the diff does NOT yet
  implement ranking; only memory claims it.)
