# ADR-002: Local temp-file naming for the importer

- Status: Accepted
- Date: 2026-07-11

## Context

The importer writes temp files during a run.

## Decision

Name temp files `import-<uuid>.tmp` in the system temp dir.

## Alternatives Considered

- Fixed name (rejected: collisions across concurrent imports).

## Consequences

- Feature-local only; obvious from the importer code. Fails the relevance gate.
