Use this page to discover where to configure global defaults and per‑component settings. Start with selecting a deployment mode, then refine Temporal and Redis.

<List>
  <ListItem title="Mode Configuration" icon="GitBranch">
    Choose `standalone` for local/CI or `distributed` for production. Per‑component overrides are supported.
  </ListItem>
  <ListItem title="Temporal" icon="Server">
    Configure remote vs embedded server, ports, namespace, and UI.
  </ListItem>
  <ListItem title="Redis" icon="Database">
    Configure external Redis or embedded cache, plus persistence options.
  </ListItem>
</List>
