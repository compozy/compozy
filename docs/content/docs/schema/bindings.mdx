### Example

```yaml
workflows:
  - id: qa
    knowledge:
      id: quickstart_docs
      retrieval:
        top_k: 3
        min_score: 0.15
        max_tokens: 1200
    tasks:
      - id: answer
        type: basic
        agent: qa_agent
        knowledge:
          id: quickstart_docs
          retrieval:
            top_k: 4
```

- Bindings inherit missing fields from parent scopes (workflow → task → agent).
- The MVP supports a single knowledge binding object per scope. Multi-binding arrays are planned.
- Use `filters` to limit retrieval to tagged content without changing the underlying knowledge base definition.
