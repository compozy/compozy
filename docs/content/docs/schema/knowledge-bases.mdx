### Example

```yaml
knowledge_bases:
  - id: quickstart_docs
    embedder: openai_default
    vector_db: in_memory_faststart
    sources:
      - kind: markdown_glob
        glob: "docs/**/*.md"
    chunking:
      strategy: token
      size: 512
      overlap: 64
    retrieval:
      top_k: 5
      min_score: 0.2
      filters:
        tag: "policy"
```

- `sources` supports `markdown_glob` and `pdf_url` in the MVP. Additional kinds appear when feature flags are enabled.
- Omit `size`, `overlap`, or `top_k` to inherit defaults from `config.knowledge`.
- Use metadata under `sources[].metadata` to attach tags for filtered retrieval.
