# Tasks Overview

Tasks are the fundamental building blocks of Compozy workflows. They represent individual operations that can be executed as part of your AI-powered applications. Each task type is designed for specific use cases, from simple operations to complex orchestration patterns.

## Getting Started

New to Compozy tasks? Start with [Basic Tasks](/docs/core/tasks/basic-tasks) to understand the fundamentals, then explore [Flow Control](/docs/core/tasks/flow-control) and [Parallel Processing](/docs/core/tasks/parallel-processing) to build more sophisticated workflows.

For production deployments, review [Advanced Patterns](/docs/core/tasks/advanced-patterns) to learn enterprise-grade orchestration techniques and best practices.
